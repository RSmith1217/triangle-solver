# Triangle Lab

An interactive, responsive triangle solver for SSS, SAS, ASA, AAS, and SSA cases.

## Features

- Laws of Sines and Cosines
- Proportional SVG triangle visualization
- Ambiguous SSA case support
- Area and perimeter calculations
- Responsive classroom-friendly interface

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
